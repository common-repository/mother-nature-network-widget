== Description ==

Display the top stories from the Mother Nature Network (<a href="http://www.mnn.com">MNN.com</a>) in your sidebar. 


== Installation ==

1. Download 'MNN-top-stories.zip' to your computer and unzip it.
2. Upload 'MNN-top-stories' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the widget to your sidebar by clicking on 'Appearances > Widgets' menu in the Wordpress admin section.
